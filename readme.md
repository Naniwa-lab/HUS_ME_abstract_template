

# 卒業論文 梗概 LaTeX テンプレート  
（北海道科学大学 工学部 機械工学科）

このテンプレートは，北海道科学大学 工学部 機械工学科における  
**卒業論文 梗概（予稿）** 作成用の LaTeX テンプレートです。

Word 版テンプレートの体裁・情報構成を踏襲しつつ，  
LaTeX 利用者が安全に執筆できるように設計されています。

---

## 1. ファイル構成

```
.
├── main.tex            ← 学生が編集するのはこのファイルのみ
├── HUS_abstract.cls    ← 学科専用クラス（編集不要）
├── jsmepaper.cls
├── a4jsme.sty
├── jsmefont.sty
├── jsme10.sty
├── jsadd.sty
└── figure/
    └── 図ファイル（.eps, .pdf など）
```

**注意**  
`HUS_abstract.cls` および `jsme*` 系ファイルは編集しないでください。  
レイアウト・書式はすべて cls 側で制御されています。

---

## 2. 編集するファイルについて

学生が編集するのは **main.tex のみ**です。

- タイトル  
- 著者情報  
- 本文  
- 図  
- 参考文献  

はすべて `main.tex` に記述してください。  
レイアウトや余白を調整する命令を書く必要はありません。

---

## 3. タイトル・著者情報の書き方

```tex
\\titleja{和文タイトル}
\\titleen{English Title}

\\studentid{学籍番号}
\\authorja{氏名}
\\advisor{指導教員名}

\\authoren{Author Name}

\\keywords{keyword1, keyword2, keyword3}
```

---

## 4. 本文の構成について

章構成は **研究分野・内容に応じて自由に設定して構いません**。  
本テンプレートでは，章の数や名称を制限していません。

ただし，卒業論文 梗概として，以下の内容を含めることを推奨します。

- 研究の背景・目的  
- 手法（実験方法，解析方法など）  
- 結果および考察  
- 結論またはまとめ  

### 構成例（あくまで一例）

```tex
\\section{緒言}
\\section{手法}
\\section{結果}
\\section{考察}
\\section{結言}
```

または，

```tex
\\section{緒言}
\\section{実験方法}
\\section{実験結果および考察}
\\section{結言}
```

※ 上記は参考例であり，必須ではありません。  
※ 研究内容に応じて，適切な構成を選択してください。

---

## 5. 図の入れ方

```tex
\\begin{figure}[b]
  \\centering
  \\includegraphics[width=0.9\\linewidth]{figure/sample.pdf}
  \\caption{図のキャプション}
\\end{figure}
```

- 図番号は自動的に **「図 1」** の形式で表示されます。  
- 1ページ目の上部に図が配置されない仕様になっています。

---

## 6. 参考文献について（重要）

### 本文中での引用

本文中の引用は，**必ず `\\cite{}` を使用してください**。

```tex
先行研究により報告されている\\cite{ref1}．
```

出力例：

```
……報告されている(1)
```

※ 手書きで (1) を入力しないでください。

---

### 参考文献リスト

```tex
\\begin{thebibliography}{99}

\\bibitem{ref1}
著者名：論文タイトル，雑誌名，巻，ページ（年）．

\\bibitem{ref2}
Author, A. B., Title of the paper, Journal name, Vol., pp. (Year).

\\end{thebibliography}
```

- 文献番号は **本文で引用された順**に自動で付与されます。  
- 文献リストでは `(1), (2), …` の形式で表示されます。

## 7. コンパイル方法

以下のいずれかでコンパイルできます。

```bash
platex main.tex
dvipdfmx main.dvi
```

または，

```bash
latexmk main.tex
```

## 8. 本テンプレートの設計方針（補足）

- 日本機械学会（JSME）テンプレートは改変していません。  
- 学科固有の仕様は `HUS_abstract.cls` にてラップしています。  
- 学会原稿用テンプレートとは独立して利用できます。

---

## 9. 注意事項

- BibTeX は使用しません。  
- レイアウト調整用の命令（`\\vspace` など）は使用しないでください。  
- Word 版テンプレートと同等の体裁になるよう設計されています。
